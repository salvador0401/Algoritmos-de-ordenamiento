# Algoritmos de ordenamiento Externo
  1.	Straight Merging:
    •	Promedio: O(n log n)
    •	Mejor Caso: O(n log n)
    •	Peor Caso: O(n log n)
  2.	Natural Merging:
    •	Promedio: O(n log n)
    •	Mejor Caso: O(n log n)
    •	Peor Caso: O(n log n)
  3.	Balanced Multiway Merging:
    •	Promedio: O(n log n)
    •	Mejor Caso: O(n log n)
    •	Peor Caso: O(n log n)
  4.	Polyphase Sort:
    •	Promedio: O(n log n)
    •	Mejor Caso: O(n log n)
    •	Peor Caso: O(n log n)
